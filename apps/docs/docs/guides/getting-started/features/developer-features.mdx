# Features for Developer

Selected features for developing Tasenor Bookkeeper:

* **Versatile Plugins** — Various plugin types for convenient extension in any direction.
* **Wide Range of Tests** — Comprehensive private and public tests on real-life scenarios to ensure changes do not cause disruptions.
* **Latest Tech** — Continuous progress towards the latest web development stack.
* **Developer Oriented** — The system follows a developer-first approach. Once launched, the development environment instantly displays all changes (incl. UI, backend and documentation) on screen.
