# Features for Developer

Few selected features for developing the Tasenor Bookkeeper.

* **Versatile Plugins** — Multiple different plugin types allowing easy extension to any direction needed.
* **Wide Range of Tests** — Remarkable collection of public and private tests with real world test cases to ensure that changes introduced does not break existing things.
* **Latest Tech** — Constantly driving toward the latest development stack for web-environment.
* **Developer Oriented** — All parts of the system has been set up in developer first principle. Launch the development environment and all changes are immediately reflected on screen. That includes UI, backend and documentation.
