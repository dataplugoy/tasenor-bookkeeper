# User Install

Installing local Tasenor Bookkeeper for actual use.

## Prerequisities

Docker is the only requirement.

* [docker](https://www.docker.com/get-started/)

## Installing and Running

Get a copy of the repository
```sh
git clone https://github.com/dataplugoy/tasenor-bookkeeper.git
```
Build and run
```sh
cd tasenor-bookkeeper/
docker compose up
```

Now, create an admin user and start exploring the system at [http://bookkeeper.localhost/](http://bookkeeper.localhost/).
