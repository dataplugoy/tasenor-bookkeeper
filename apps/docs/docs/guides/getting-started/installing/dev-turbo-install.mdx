# Local Turbo Development

Steps to set up Tasenor Bookkeeper for local development use.

## Prerequisites

To install Tasenor Bookkeeping, the following are needed:

* Node environment (prefereably with [nvm](https://github.com/nvm-sh/nvm))
* [pnpm](https://pnpm.io/)
* [turbo](https://turbo.build/repo)
* [jq](https://jqlang.github.io/jq/)
* [docker](https://www.docker.com/get-started/)

## Installing Turbo Development Copy

Get a copy of the repository
```sh
git clone https://github.com/dataplugoy/tasenor-bookkeeper.git
```
Then install needed NPM-packages
```sh
cd tasenor-bookkeeper/
pnpm install
```

## Launching

Now, launch it in developer mode with
```sh
turbo dev
```
Next, create an admin user and start exploring the system at [http://localhost:7204/](http://localhost:7204/).
In this mode the changes in the code are reflected immediately.
Documentation is available in [http://localhost:7207/](http://localhost:7207/).
